## Deployment Configuration
Uygulama, ortam bazlı yapılandırmalarını `appsettings.{Environment}.json` dosyalarından ve gizli değerler için kullanıcı gizli yöneticisi, ortam değişkenleri veya güvenli dosyalardan okur. Bu nedenle, hassas veriler (veritabanı bağlantı dizeleri, yönetici parolaları, Iyzico anahtarları vb.) sürüme dahil edilmemeli ve dağıtım sırasında ilgili ortama enjekte edilmelidir.
